Página de vendas alura

Para este projeto de página de vendas, meu foco foi criar uma interface simples, mas visualmente atraente, usando HTML e CSS, com um toque de personalização adicional por meio do Google Fonts. A ideia era construir uma página que apresentasse de forma clara e persuasiva um produto específico, com uma estrutura de layout que guiasse o visitante pela descrição e pelos benefícios do produto. Comecei estruturando o HTML para que cada seção tivesse um propósito bem definido: uma introdução do produto no topo, seguida por uma descrição detalhada e depoimentos de clientes. Na seção de introdução, usei um título e um subtítulo chamativos, seguidos por uma imagem grande do produto que destacasse suas principais qualidades. Para manter o layout bem organizado e intuitivo, apliquei CSS flexbox e grid, permitindo que a página fosse responsiva e se adaptasse bem a diferentes tamanhos de tela. Para a parte visual, escolhi uma paleta de cores moderna e contrastante, usando tons que transmitissem confiança e profissionalismo. Decidi por uma fonte personalizada do Google Fonts para o título e subtítulo, que adicionasse um toque especial ao design e criasse um visual mais marcante. Além disso, ajustei o CSS para incluir animações suaves, como efeitos de hover nos botões e transições de cor nos títulos, para tornar a experiência mais dinâmica e interativa. O resultado final foi uma página de vendas bem estruturada e envolvente. Fiquei satisfeito em ver como o uso de HTML e CSS pode transformar uma ideia em uma página funcional e atrativa, e aprendi bastante sobre a importância da tipografia e do design responsivo na criação de interfaces web.

Site pessoal

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, e JavaScript, que simula uma experiência narrativa centrada no tema da Inteligência Artificial. O objetivo é engajar os jogadores em um cenário fictício que aborda o avanço da IA e suas possíveis consequências para a humanidade, destacando a importância da reflexão ética sobre a tecnologia. Neste projeto de site pessoal, utilizei HTML, CSS e Google Fonts para criar uma plataforma que apresentasse minhas informações e interesses de maneira organizada e visualmente atraente. A ideia era desenvolver uma página que pudesse funcionar como meu cartão de visita digital, onde qualquer pessoa pudesse conhecer mais sobre quem eu sou, meus projetos, habilidades e interesses pessoais e profissionais. Comecei planejando a estrutura em HTML, dividindo a página em seções como “Sobre Mim”, “Habilidades”, “Projetos”, “Interesses” e “Contato”. A seção “Sobre Mim” é o ponto de entrada, onde destaquei uma breve introdução com uma foto e uma descrição sobre minha formação e objetivos. Em “Habilidades”, organizei as principais tecnologias e ferramentas que domino, enquanto na seção “Projetos” listei alguns dos trabalhos que desenvolvi, com links para repositórios ou demonstrações para quem quiser ver mais a fundo. Para a parte estética, usei CSS para criar um layout limpo e agradável. Escolhi uma paleta de cores que combinasse com minha personalidade, optando por tons neutros com alguns detalhes de cor para dar destaque às seções importantes. As seções foram dispostas em um layout responsivo, utilizando flexbox e grid, garantindo que o site se adaptasse bem a qualquer dispositivo, seja desktop, tablet ou smartphone. Um toque especial foi a tipografia personalizada, para a qual escolhi fontes do Google Fonts. Usei uma fonte de destaque nos títulos para dar uma identidade própria ao site e uma fonte mais leve e legível para os textos, criando uma harmonia visual que facilita a leitura. Além disso, adicionei algumas animações suaves, como efeitos de hover nos botões e transições nas imagens, que ajudam a criar uma experiência mais interativa e fluida para o visitante. O projeto me ensinou muito sobre a importância do design responsivo e da tipografia na construção de sites pessoais. Além disso, foi uma ótima oportunidade para refletir sobre como me apresentar de forma clara e autêntica online, escolhendo cuidadosamente os elementos e o conteúdo que transmitissem quem eu sou e no que acredito.
